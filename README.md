# dev-academy-spring-2026-api

This is an API implementation for Solita Dev Academy Finland March 2026 pre-assignment.

The requirements for the assignment as well as the instructions for running the database can be found at  
https://github.com/solita/dev-academy-spring-2026-exercise

# Instructions for running the API
1. Make sure the database is up and runnning
2. Clone this repository
3. At the root of the project run:  
```
mvn spring-boot:run
```

# UI
The repository for the UI implementation using this API can be found at  
https://github.com/m4rkusmk/dev-academy-spring-2026-ui
